# Word2Vec (Skip-Gram with Negative Sampling) — NumPy Implementation

This repository contains a **implementation of Word2Vec (SGNS)** using Numpy  
(no PyTorch / TensorFlow or other ML frameworks).

---

## Features
- Skip-Gram with Negative Sampling (SGNS)
- NumPy implementation (forward, backward, SGD updates)
- Unigram negative sampling with power 0.75
- Modular project structure (preprocessing, dataset, model, training)
- Reproducible training with fixed random seed

---

## Project Structure
train_word2vec_sgns_evaluate.ipynb contains a complete end-to-end walkthrough of the Word2Vec (SGNS) pipeline, from text preprocessing to training and evaluation.
