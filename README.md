# Word2Vec (Skip-Gram with Negative Sampling) — NumPy Implementation

This repository contains an implementation of Word2Vec (SGNS) using Numpy  
(no PyTorch / TensorFlow or other ML frameworks).

---

## Features
- Skip-Gram with Negative Sampling (SGNS)
- NumPy implementation (forward, backward, SGD updates)
- Unigram negative sampling with power 0.75

---

## Project Structure

train_word2vec_sgns_evaluate.ipynb covers all main stages of the Word2Vec (SGNS) pipeline, including data preparation, model training, and evaluation.
